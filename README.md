## Setup

1. Install dependencies

   ```sh
   pnpm install
   ```

2. Start the dev server

   ```sh
   pnpm run dev
   ```

## Build

1. Generate a static production build

   ```sh
   pnpm build
   ```

2. Preview the production build locally

   ```sh
   pnpm preview
   ```
